# CardioNerve

CardioNerve is a web-based cardiac health analysis platform designed to visualize heart-related metrics and provide intelligent risk insights through a clean, modern interface.

Live application: https://cardionerve.vercel.app/

---

## Overview

Cardiovascular diseases often develop silently over time. CardioNerve aims to improve early awareness by presenting cardiac indicators in a structured and accessible format. The platform emphasizes clarity, performance, and scalability, serving as a foundation for future AI models, wearable integrations, and backend expansion.

---

## Features

- Cardiac health data visualization  
- Heart risk analysis logic  
- Responsive and user-friendly interface  
- Modular frontend architecture  
- Cloud-native deployment on Vercel  

---

## Local Setup

```bash
git clone https://github.com/abbysallord/Cardionerve.git
cd Cardionerve
npm install
npm run dev
```
Open http://localhost:3000 in your browser.
